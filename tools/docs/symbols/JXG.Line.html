Class JXG.Line


Extends JXG.GeometryElement.
Creates a new basic line object. Do not use this constructor to create a line. Use JXG.Board#create with type Line, Arrow, or Axis instead.

{float} X(phi)
Treat the line as parametric curve in homogeneuous coordinates. 
x = 1 * sin(theta)*cos(phi)
y = 1 * sin(theta)*sin(phi)
z = 1 * sin(theta)
and the line is the set of solutions of a*x+b*y+c*z = 0. It follows: 
sin(theta)*(a*cos(phi)+b*sin(phi))+c*cos(theta) = 0
Define: 
  A = (a*cos(phi)+b*sin(phi))
  B = c
Then 
cos(theta) = A/sqrt(A*A+B*B)
sin(theta) = -B/sqrt(A*A+B*B)
and X(phi) = x from above. phi runs from 0 to 1
