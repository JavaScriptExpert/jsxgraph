Class JXG.Board

This is the Board class. It stores all methods and properties required to manage a geonext board like adding geometric elements, removing them, managing mouse over, drag & drop of geometric objects etc. You should never create a board by calling this constructor. Please use JXG.JSXGraph#initBoard instead.
Defined in: Board.js.
