JavaScript: 
  - Interpreted language, JIT compilers evolving
  - Graphics: SVG (Firefox 2.0, Chrome, Safari 3, Opera, iPhone 2.1), VML (IE 5)
  - The World's Most Misunderstood Programming Language (Douglas Crockford)

JSXGraph: library for interactive Geometry/Mathematics in the web browser  
Supported browsers: Firefox 2.0, IE 6, Safari 3, Chrome, Opera

Features: 
  - Dynamic geometry: Euclidean and homogeneous coordinates
  - Curves: Function graphs, polar plots, parametric curves, data plots
  - Turtle graphics
  
Why JavaScript?
  - Canonical scripting language - JavaScript
  - Completely integrated in the web page with dynamic access to all HTML elements
  - Zero initialization time 
  - Language features of JavaScript enhance the interactivity
  - No web server required
  - Simple server access via AJAX
  
JavaScript: Functions are first order variables
