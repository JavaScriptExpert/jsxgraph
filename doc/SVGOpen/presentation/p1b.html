How can we do graphics in the web browser without Java and Flash?

  • Vector graphics for the display:
    • SVG (Firefox 2.0+, Chrome, Safari 3+, Opera, iPhone 2.1+)
    • VML (IE 5+)
  • JavaScript for the programming:
    • GeoSVG (http://wme.cs.kent.edu/geosvg),
    • JSXGraph (http://jsxgraph.org)
